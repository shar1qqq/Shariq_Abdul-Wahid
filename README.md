# Shariq Abdul-Wahid — Personal Portfolio

A personal portfolio website built with **Node.js**, **Express**, and **EJS** templating. Features a clean, responsive design with custom CSS, scroll animations, and a multi-page layout.

🌐 **Live Site:** https://shariq-abdul-wahid.onrender.com/

## About

## Running Locally

**1. Install Node.js**
Download the LTS version from https://nodejs.org

**2. Clone the repo**
```bash
git clone https://github.com/shar1qqq/Shariq_Abdul-Wahid.git
cd Shariq_Abdul-Wahid
```

**3. Install dependencies**
```bash
npm install
```

**4. Start the server**
```bash
node server.js
```

**5. Open in browser**
http://localhost:3000
